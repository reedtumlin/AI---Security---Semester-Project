The APRICOT Dataset
============================================================
A collection of Adversarial Patches Rearranged In COnText

Annotation files are provided in a .json format compatible
with the COCO API. This includes bounding box positions for
each adversarial patch in each image, along with additional
metadata such as patch target class, target model, estimated
angle and warping.
